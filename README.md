# Movie Recommendation System

A web-based movie recommendation application built with Python and Flask.

## Project Structure

- `app.py`: Main application file containing the Flask server logic.
- `requirements.txt`: List of Python dependencies required to run the application.
- `static/`: Directory for static assets.
  - `script.js`: JavaScript file for client-side functionality.
  - `style.css`: CSS file for styling the web pages.
- `templates/`: Directory for HTML templates.
  - `foryou.html`: Template for personalized recommendations.
  - `index.html`: Main landing page template.
  - `recommend.html`: Template for general movie recommendations.

## Installation

1. Ensure Python is installed on your system.
2. Clone or download the project files.
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the application:
```
python app.py
```

Open your web browser and navigate to `http://localhost:5000` (or the specified port) to access the application.

## Features

- Personalized movie recommendations
- General movie suggestions
- Web interface for easy interaction
